Introduction

Python is an interpreted, high-level, general-purpose programming language.

It supports multiple programming paradigms, including structured, object-oriented, and functional programming.

Python is dynamically typed and garbage-collected.

Its design philosophy emphasizes code readability with its use of significant indentation.

print("Hello, world!")
Variables

Variables in Python are created when you assign a value to them.

You don't need to declare the type of a variable beforehand.

Variable names are case-sensitive and must follow certain rules.

x = 5 name = "Alice"
  • Must begin with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Are case-sensitive
Data Types

Python has several built-in data types.

Some common ones include integers, floats, strings, and booleans.

You can check the type of a variable using the `type()` function.

type(5) type("hello")
  • int
  • float
  • str
  • bool
  • list
Control Flow

Control flow tools include if statements, for and while loops.

They let you execute code conditionally or repeatedly.

The `if` statement is used for conditional execution.

if x > 0: print("Positive")
Functions

Functions are defined using the `def` keyword.

They help you reuse code and organize programs logically.

You can also pass parameters and return values.

def greet(name): return "Hello " + name
  • Use `def` to define a function
  • Functions can return a value
  • Functions can have default arguments
  • Functions can take *args and **kwargs
  • Functions improve code modularity
